# 🍺 Homebrew Tap

## Installation

```sh
brew tap epilande/tap
```

## Formulas

### [codegrab](https://github.com/epilande/codegrab)

An interactive CLI tool for selecting and bundling code into a single, LLM-ready output file.

```sh
brew install epilande/tap/codegrab
```

### [repos](https://github.com/epilande/repos)

A CLI tool for managing multiple git repositories.

```sh
brew install epilande/tap/repos
```
